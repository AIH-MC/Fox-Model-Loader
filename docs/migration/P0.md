# P0 NeoForge Skeleton

This project is the standalone NeoForge migration target for Fox Model Loader.

Completed:

- Created an independent Gradle project outside the Fabric workspace.
- Added ModDevGradle configuration for NeoForge.
- Added a minimal `@Mod` entrypoint.
- Added `META-INF/neoforge.mods.toml`.
- Added an empty `yes_steve_model.mixins.json`.
- Resolved the NeoForge dependency coordinate to `26.1.2.70-beta`; the shorter `26.1.2` coordinate is not published on the NeoForge Maven.
- Removed strict `FAIL_ON_PROJECT_REPOS` repository mode because ModDevGradle adds Mojang repositories while applying the plugin.
- Added project-local acceleration settings: Aliyun Maven mirrors, BMCLAPI Minecraft libraries mirror, BMCLAPI launcher manifest, Gradle caching, and NeoFormRuntime Eclipse compiler mode.
- Enabled ModDevGradle `disableRecompilation` for the P0 skeleton to skip the slow decompile/recompile pipeline. Turn it off later if attached Minecraft sources are needed.
- Added `local-maven` as the first repository for manually downloaded NeoForge/NeoForm/ModDevGradle artifacts.

Deferred:

- Copying `common/src/main/java`.
- Copying `common/src/main/resources`.
- Migrating Fabric API, Cardinal Components API, Architectury, and Forge Config API Port code.
