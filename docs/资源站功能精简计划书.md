# 资源站功能精简计划书

> 项目：`D:\OYSM\openysm-26.1.2`  
> 范围：客户端资源站 UI、远程列表、下载、导入、上传、服务端重载同步  
> 结论：可行；必须复用现有上传链路，不能只写客户端本地 `custom`。

## 一、目标

在模型选择界面增加“资源站”，支持浏览远程模型资源，下载 `.ysm` / `.zip`，并复用现有导入、上传、服务端重载、客户端同步链路。

## 二、现状依据

当前项目已有：

```text
PlayerModelScreen：模型选择界面，可增加入口
DownloadScreen：已有 Coming Soon 占位页
ModelUploadScreen：已有导入 UI、进度、错误状态参考
ModelUploadSession：已有客户端上传会话
ServerModelManager.finishModelUpload：已有服务端校验、写入 CUSTOM、重载、同步
ClientModelManager.importLocalModel / reloadLocalModels：已有本地导入和扫描能力
GeneralConfig / ModConfigSpec：已有配置模式
```

## 三、核心原则

不要把“下载到客户端本地 custom 目录”作为主流程。

正确主流程：

```text
资源站下载 bytes
-> 本地校验/预加载
-> ModelUploadSession.start(modelId, fileName, bytes)
-> 服务端 finishModelUpload
-> 服务端写入 config/yes_steve_model/custom
-> 服务端 reload
-> 服务端同步给客户端
```

原因：

```text
单机：客户端目录和服务端目录通常是同一环境，可以生效。
远程服务器：客户端本地目录不是服务器目录，服务端扫描不到。
```

## 四、实施阶段

### P0：最小可用

1. 新增 `ResourceStationScreen`。
2. `DownloadScreen` 改为 `ResourceStationScreen` 的兼容入口。
3. `PlayerModelScreen` 增加资源站按钮。
4. 新增 `ResourceStationConfig`，保存默认 URL、当前选中 URL、超时、最大下载大小。
5. 新增 `ModelRepoEntry`。
6. 新增 `ModelRepoClient`。
7. 支持 OpenYSM `index.json`。
8. 支持 `.ysm` / `.zip` 下载。
9. 下载、列表刷新放后台线程。
10. UI 更新切回 Minecraft 主线程。
11. 下载完成后调用 `ModelUploadSession.start(...)`。
12. 界面监听 `ModelUploadSession.Listener` 显示上传状态。
13. 增加 `zh_cn` / `en_us` 语言键。

### P1：GitHub 兼容

支持：

```text
https://github.com/{owner}/{repo}
https://github.com/{owner}/{repo}/tree/{branch}
https://github.com/{owner}/{repo}/tree/{branch}/{path}
```

要求：

```text
只接受 https://github.com
使用 GitHub Contents API，不解析 HTML
仓库根地址优先读取默认分支，或第一版先尝试 main
递归列出 .ysm / .zip
处理 GitHub API 错误和速率限制
```

默认资源站：

```text
https://github.com/Elaina69/Yes-Steve-Model-Repo/tree/main
https://github.com/sdf123098/YSM-Model
```

4. 预览图、作者、标签、描述。

## 五、安全要求

```text
禁止路径穿越：../、..\
禁止绝对路径
modelId 只允许 [a-z0-9_./-]+
中文、日文、空格文件名只作为显示名
无法生成安全 modelId 时使用稳定 hash 兜底
限制最大下载体积
校验扩展名
关闭界面后旧后台任务不得回写已关闭界面
```

推荐兜底：

```text
显示名：绯英_0.4.ysm
modelId：repo/sdf123098/ysm-model/<sha1前12位>
上传文件名：<sha1前12位>.ysm
```

## 六、验收标准

```text
模型选择界面能打开资源站
资源站能返回模型选择界面
默认 URL 可见且可编辑
index.json 能列出 .ysm / .zip
GitHub 根地址和 tree URL 能列出 .ysm / .zip
下载不阻塞主线程
导入走 ModelUploadSession
远程服务器场景不会误写客户端本地 custom
成功后模型列表自动刷新
非法 URL、网络失败、下载失败、导入失败都有提示
```

## 七、明确不做

```text
不做 GitHub 登录/token
不做评分评论收藏
不做断点续传
不做文件夹模型递归打包下载
不新增 Mixin/Coremod/ASM
```
